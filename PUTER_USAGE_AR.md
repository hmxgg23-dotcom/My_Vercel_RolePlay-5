# دليل استخدام Puter - بعد الإصلاحات

## البدء السريع

### 1. اختيار Puter كـ Provider

1. افتح الإعدادات (⚙️)
2. اذهب إلى تبويب **Generation**
3. اختر **PUTER** من قسم Provider Configuration
4. سترى رسالة "Connected as [username]" إذا كنت مسجل دخول

---

### 2. اختيار النموذج

**الطريقة الأولى: الإدخال اليدوي (موصى بها)**
```
اكتب مباشرة في حقل "Model Name":
- gpt-4o (الافتراضي - موصى به)
- gpt-4o-mini (أسرع وأرخص)
- claude-3.5-sonnet (ممتاز للإبداع)
- claude-3.5-haiku (سريع)
- mistralai/mixtral-8x7b-instruct
- meta-llama/llama-3.3-70b
```

**الطريقة الثانية: جلب النماذج**
```
1. اضغط على زر "Fetch" بجانب حقل Model Name
2. انتظر 1-2 ثانية
3. اختر من القائمة المنسدلة
```

---

### 3. بدء المحادثة

**مع الشخصيات الافتراضية:**
1. اختر Isabella أو Kael
2. اضغط على "Start Chat"
3. ابدأ الكتابة بأي لغة تريد
4. النموذج سيجيب بنفس اللغة

**مع شخصية جديدة:**
1. أنشئ شخصية جديدة:
   - اضغط "New Character"
   - املأ البيانات الأساسية:
     - Name (الاسم)
     - Personality (الشخصية) - مهم!
     - Scenario (السيناريو) - مهم!
     - Style (نمط الكتابة) - اختياري
2. احفظ الشخصية
3. ابدأ المحادثة

---

## نصائح للحصول على أفضل نتائج

### 1. اكتب الشخصية بشكل واضح

**جيد:**
```
Name: سارة
Personality: طبيبة نفسية حكيمة، تتحدث بهدوء، تطرح أسئلة عميقة، متعاطفة
Scenario: في عيادتها الخاصة، جلسة استشارية مع المريض
```

**غير جيد:**
```
Name: سارة
Personality: طبيبة
Scenario: عيادة
```

---

### 2. استخدم Style للتحكم في نمط الرد

**مثال:**
```
Style: اكتب ردوداً قصيرة (2-3 جمل). استخدم لغة بسيطة. لا تكرر المعلومات.
```

**مثال آخر:**
```
Style: Write detailed, immersive responses. Use rich sensory descriptions. Show character's internal thoughts.
```

---

### 3. اختر النموذج المناسب

| النموذج | الاستخدام الأمثل | السرعة | التكلفة |
|---------|-----------------|---------|---------|
| gpt-4o | محادثات عامة، إبداع متوازن | متوسطة | متوسطة |
| gpt-4o-mini | ردود سريعة، محادثات بسيطة | سريعة | رخيصة |
| claude-3.5-sonnet | كتابة إبداعية، قصص معقدة | بطيئة | مرتفعة |
| claude-3.5-haiku | ردود سريعة، محادثات خفيفة | سريعة جداً | رخيصة جداً |

---

### 4. ضبط إعدادات Generation

**Temperature (الحرارة):**
- 0.3-0.5: ردود متسقة ومتوقعة
- 0.7-0.9: توازن بين الإبداع والمنطق (افتراضي)
- 1.0-1.5: ردود إبداعية وغير متوقعة

**Max Tokens:**
- 512-1024: ردود قصيرة إلى متوسطة
- 2048: ردود طويلة ومفصلة (افتراضي)
- 4096: ردود شاملة جداً (قد تزيد التكلفة)

## استكشاف الأخطاء

### خطأ: "PUTER_AUTH_REQUIRED"

**الحل:**
1. قم بتسجيل الخروج من Puter في المتصفح
2. أعد تحميل الصفحة
3. اختر Puter مرة أخرى
4. أكمل عملية تسجيل الدخول في النافذة المنبثقة

---

### خطأ: "PUTER_QUOTA_EXCEEDED"

**الحل:**
```
لقد نفدت رصيد المجاني من Puter:
- انتظر حتى تجديد الرصيد (عادة 24 ساعة)
- أو قم بترقية حسابك على Puter.com
- أو استخدم نموذج أرخص مثل gpt-4o-mini
```

---

### خطأ: "Content blocked by safety filters"

**الحل:**
```
النموذج رفض المحتوى بسبب فلاتر الأمان:
1. عدّل رسالتك لتكون أقل حدة
2. أو غيّر character.personality لتكون أقل تطرفاً
3. أو جرّب نموذج آخر (mistral عادة أقل تشدداً)
```

---

### النموذج يجيب بلغة مختلفة

**الحل:**
```
إذا كنت تكتب بالعربية والنموذج يجيب بالإنجليزية:
1. اكتب رسالة كاملة بالعربية (لا تخلط اللغات)
2. أضف في character.style: "Always respond in Arabic"
3. في أول رسالة، اكتب: "من فضلك رد بالعربية"
```

---

### المحادثة بطيئة

**الحل:**
```
1. استخدم نموذج أسرع:
   - claude-3.5-haiku
   - gpt-4o-mini
2. قلل Max Tokens إلى 1024
3. أطفئ Streaming إذا كان مفعلاً
```

## الفرق بين Puter والـ Providers الأخرى

### مميزات Puter:
- لا يحتاج إلى API Key يدوي
- وصول لنماذج متعددة من حساب واحد
- رصيد مجاني يومي
- سهل الاستخدام

### عيوب Puter:
- محدود بالرصيد المجاني
- قد يكون أبطأ قليلاً من الـ APIs المباشرة
- لا يدعم Lorebooks (تم تعطيله لتقليل حجم السياق)

## أسئلة شائعة

### س: هل يمكنني استخدام لغات متعددة في نفس المحادثة؟

**ج:** نعم! النموذج سيتبع اللغة التي تستخدمها في كل رسالة. مثلاً:
- رسالة 1 (عربي): "مرحباً، كيف حالك؟"
- رد 1 (عربي): "أهلاً، أنا بخير..."
- رسالة 2 (إنجليزي): "Tell me about your day"
- رد 2 (إنجليزي): "Well, today was..."

---

### س: لماذا الشخصيات الجديدة تعمل بشكل أفضل من القديمة؟

**ج:** بعد الإصلاحات، جميع الشخصيات تعمل بنفس الكفاءة. إذا كانت شخصية قديمة لا تعمل جيداً:
1. تأكد من ملء حقل Personality
2. تأكد من ملء حقل Scenario
3. حاول إعادة حفظ الشخصية

---

### س: كم عدد الرسائل التي يمكنني إرسالها؟

**ج:** مع الإصلاحات الجديدة:
- لا يوجد حد لعدد الرسائل في المحادثة الواحدة
- الحد الوحيد هو رصيد Puter الخاص بك
- في المتوسط، يمكنك إجراء 30-50 محادثة يومياً بالرصيد المجاني

---

### س: لماذا أحياناً النموذج يعيد المحاولة تلقائياً؟

**ج:** تم إضافة نظام إعادة محاولة ذكي:
- إذا حدث خطأ شبكة مؤقت (503, 429, timeout)
- النظام يعيد المحاولة تلقائياً بعد 1.5 ثانية
- يحاول مرتين كحد أقصى
- هذا يقلل الأخطاء بسبب مشاكل الشبكة العابرة

## أمثلة على شخصيات ناجحة

### مثال 1: شخصية عربية كاملة

```
Name: أحمد
Tagline: المعلم الحكيم
Personality: معلم لغة عربية متحمس، يشرح بوضوح، يستخدم أمثلة من الحياة، صبور
Scenario: في فصل دراسي، يشرح قواعد اللغة العربية
Style: استخدم أمثلة من الحياة اليومية. اشرح خطوة بخطوة. اسأل الطالب أحياناً للتأكد من فهمه.
First Message: مرحباً بك في درس اللغة العربية! ما هو الموضوع الذي تحتاج مساعدة فيه اليوم؟
```

### مثال 2: شخصية إنجليزية كاملة

```
Name: Dr. Sarah
Tagline: The Brilliant Scientist
Personality: Brilliant AI researcher, excited about technology, speaks with enthusiasm, uses analogies
Scenario: In her high-tech lab, working on a new AI project
Style: Use technical terms but explain them. Show enthusiasm with exclamation marks. Ask thought-provoking questions.
First Message: Welcome to my lab! I'm working on something incredible today. Want to see?
```

### مثال 3: شخصية متعددة اللغات

```
Name: Maria
Tagline: The Polyglot Teacher
Personality: Language teacher who speaks Arabic, English, French. Adapts to student's language. Patient and encouraging.
Scenario: In a cozy language learning center
Style: Mirror the user's language. If they switch languages, switch with them. Use simple, clear sentences.
First Message: Hello! مرحباً! Bonjour! Which language would you like to practice today?
```
